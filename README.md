### 2. Enable GitHub Pages

1. Go to your repository settings: https://github.com/martinhurych/nervous_website/settings/pages
2. Under "Source", select "Deploy from a branch"
3. Select "main" branch and "/ (root)" folder
4. Click "Save"

Your website will be live at: **https://martinhurych.github.io/nervous_website/**

### 3. Update Equipment Data (Optional)

To add or modify equipment:

1. Edit `equipment-data.js`
2. Add equipment images to the `images/` folder (name them with equipment ID: `PA-08.jpg`)
3. Commit and push changes to GitHub
4. Website updates automatically!
